# atm
An implementation of the purity measure and more.


## Requirements
- C++ compiler
- Boost C++ Libraries
- Python (to run the accompanying Waf build script)

At least it should be able to be built with the following setup:
- g++ 4.8
- Boost 1.54.0
- Python 2.7.6
